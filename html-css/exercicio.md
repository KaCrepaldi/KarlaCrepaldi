

# Exercício

1. Criar conta no GitHub (github.com)

2. Criar um repositório público : frontend

3. Clonar o repositório criado, exemplo:
> git clone https://github.com/KaCrepaldi/KaCrepaldi.git

4. Adicionar seus arquivos (precisa copiar todos seus arquivos do curso , exceto o .zip)
>git add*

5. Comitar seu código
> git commit -m 'commit inicial'

6. subir as alterações
> git push

6. Enviar o link para seu repositório criado na plataforma Ebac, exemplo:
https://github.com/KaCrepaldi

7. DESAFIO: criar apresentação pessoal no Github.
